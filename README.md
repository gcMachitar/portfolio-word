Student Portfolio Website
Overview

A responsive personal portfolio website built to showcase skills, services, education, experience, and social links. Features include a dark/light mode toggle, typing animation for roles, interactive social icons, and a modern, clean design.

Technologies Used:

HTML5, CSS3, JavaScript
Font Awesome for icons
Typed.js for typing animation

Features:

Responsive navigation bar
Dark/Light mode with persistent user preference
Typing animation introducing multiple roles
Interactive profile image and social icons
Call-to-action “Hire me” button

Created primarily following this YouTube tutorial: https://www.youtube.com/watch?v=-u3vE84Wo_U
